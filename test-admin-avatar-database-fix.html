✅ Implementation Applied:
// AdminAuthService.ts - Added uploadAvatar method uploadAvatar(file: File): Observable<User> { const formData = new FormData(); formData.append('file', file); const token = this.getToken(); const headers: { [key: string]: string } = {}; if (token) { headers['Authorization'] = `Bearer ${token}`; } return this.http.post<User>(`${this.API_URL}/upload-avatar`, formData, { headers }).pipe( tap((updatedUser: User) => { this.updateAdminAvatar(updatedUser); }) ); } // AdminLayoutComponent.ts - Updated to use AdminAuthService this.adminAuthService.uploadAvatar(file).subscribe({ next: (updatedUser) => { console.log('✅ Admin avatar uploaded and saved to database:', updatedUser); this.isUploadingAvatar = false; this.closeAvatarModal(); this.avatarTimestamp.set(Date.now()); this.notificationService.showSuccess('Thành công!', 'Ảnh đại diện admin đã được cập nhật thành công'); } });

🔧 Backend API Flow:

1. Frontend: AdminAuthService.uploadAvatar(file) 2. HTTP POST: /api/auth/upload-avatar (with admin JWT token) 3. Backend: AuthController.uploadAvatar(file, authentication) 4. Backend: AuthService.updateUserAvatar(file, email) 5. Database: User.avatar = new_url (saved permanently) 6. Response: UserDTO with updated avatar URL 7. Frontend: Updates localStorage + UI

  • Storage Change: localStorage only → Database + localStorage
  • Persistence Change: Session-only → Permanent
